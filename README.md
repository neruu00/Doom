# Doom-Style 2.5D Phaser Raycaster

이 프로젝트는 Phaser 3 Webpack TypeScript 템플릿을 기반으로 구현된 **둠(Doom) 스타일의 2.5D 1인칭 슈팅 게임**입니다. 
자체적인 레이캐스팅(Raycasting) 엔진을 사용하여 3D 공간을 렌더링하며, 적 AI 및 전투 시스템이 포함되어 있습니다.

## 🎮 How to Play (플레이 방법)

- **[ ↑ ] / [ ↓ ]**: 앞 / 뒤로 이동합니다.
- **[ ← ] / [ → ]**: 좌 / 우로 게걸음(스트레이프) 이동을 합니다.
- **[ Q ] / [ E ]**: 시점을 좌 / 우로 회전합니다.
- **[ W ]**: 장착된 샷건을 발사합니다. 화면 중앙의 조준점(`+`)에 적을 맞추고 쏘면 데미지를 입힐 수 있습니다.
- **[ Spacebar ]**: 게임을 일시정지 / 재개합니다.
